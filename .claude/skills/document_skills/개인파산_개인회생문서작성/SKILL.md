# 개인파산·개인회생 문서작성 스킬

## 개요

개인파산 및 개인회생은 채무자가 과도한 채무로 인해 정상적인 경제생활을 영위할 수 없을 때, 법원의 도움을 받아 채무를 청산하거나 재기할 수 있도록 하는 제도입니다. 상속재산파산은 채무초과 상태의 피상속인이 사망한 경우, 상속인이 상속재산에 대하여 파산신청을 하여 청산하는 절차입니다.

이 스킬은 75,869줄의 개인파산·상속재산파산 실무 데이터베이스를 기반으로 모든 종류의 도산 관련 신청서를 자동으로 작성합니다.

## 개인파산·개인회생 체계

### 1. 개인파산 제도

#### 1.1 파산의 의의
- **목적**: 채무초과 상태의 개인이 경제적으로 재기할 수 있는 기회 부여
- **효과**: 파산선고 → 면책허가 → 채무 면제

#### 1.2 파산 요건
- **파산원인**: 채무자가 변제기에 있는 채무를 일반적·계속적으로 변제할 수 없는 상태 (지급불능)
- **면책불허가 사유 없을 것**

#### 1.3 파산절차의 흐름
```
파산 및 면책 신청 (회생법원)
   ↓
파산선고
   ↓
파산관재인 선임 (재산 있는 경우)
   ↓
채권자집회
   ↓
면책심문
   ↓
면책허가 결정
   ↓
확정 (면책 확정)
```

### 2. 개인파산 신청서 작성

#### 2.1 신청인 정보
```
개인파산 및 면책 신청서

채무자
성명: ○○○
주민등록번호: 000000-0000000
주소: ○○시 ○○구 ○○로 123
등록기준지: ○○시 ○○구 ○○동 123
직업: ○○
전화번호: 010-0000-0000
```

#### 2.2 신청 취지
```
신청 취지

1. 채무자에 대하여 파산을 선고한다.
2. 채무자를 면책한다.

라는 결정을 구합니다.
```

#### 2.3 파산 원인
```
파산 원인

1. 채무 발생 경위
   - 2020년경부터 사업 운영 중 자금난으로 대출 받기 시작
   - 2022년 사업 부진으로 수입 감소, 채무 급증
   - 현재 총 채무액: 금 500,000,000원

2. 지급불능 상태
   - 월 수입: 약 2,000,000원 (일용직 근로)
   - 월 지출: 약 1,500,000원 (생활비, 주거비 등)
   - 여유 자금: 약 500,000원
   - 월 변제 필요액: 약 10,000,000원 (이자만)
   → 변제 불가능 상태
```

#### 2.4 채권자목록 (양식 1-2)
```
채권자목록

순번  채권자명        주소               채권종류    채권액        비고
1     ○○은행    서울 중구 ○○로    대출금    200,000,000   담보 없음
2     △△카드    서울 강남구 △△로  카드대금   50,000,000   무담보
3     □□캐피탈  부산 해운대구 □□로 할부금    30,000,000   자동차 담보
...

합계: 500,000,000원
```

#### 2.5 재산목록 (양식 1-4)
```
재산목록

[부동산]
- 없음

[동산]
- 승용차 1대 (2018년식 ○○) - 시가 5,000,000원 (할부 잔액: 30,000,000원)
- 가전제품 일체 - 시가 2,000,000원

[채권·예금]
- ○○은행 보통예금 - 잔액 500,000원

[기타 재산]
- 퇴직금 청구권 (예상액: 10,000,000원)

합계: 약 17,500,000원
```

#### 2.6 첨부서류
- [ ] 주민등록등본
- [ ] 가족관계증명서
- [ ] 소득증명서류 (근로소득 원천징수영수증, 급여명세서 등)
- [ ] 재산증명서류 (부동산등기부등본, 예금잔액증명서, 자동차등록증 등)
- [ ] 채권증빙서류 (대출약정서, 독촉장, 지급명령 등)

### 3. 면책 제도

#### 3.1 면책의 의의
- **효과**: 파산선고로 확정된 채무가 면제됨 (파산채권의 책임 면제)
- **예외**: 조세채권, 벌금, 양육비, 고의로 가한 불법행위 채권 등은 면책되지 않음

#### 3.2 면책불허가 사유 (채무자회생법 제564조)
1. **사해행위**: 파산선고 1년 전부터 재산을 은닉·훼손·감소시키는 행위
2. **허위 채권자목록 제출**: 일부 채권자를 고의로 누락
3. **허위 재산목록 제출**: 재산을 누락하거나 과소 신고
4. **낭비, 도박 등**: 사치, 도박, 투기행위로 재산 감소
5. **설명거부, 허위설명**: 법원이나 파산관재인에 대한 설명 거부 또는 거짓 설명
6. **장부 미작성, 은닉**: 장부를 작성하지 않거나 은닉한 경우
7. **7년 이내 면책**: 과거 7년 이내에 면책받은 사실이 있는 경우

#### 3.3 선면책 제도
- **의의**: 파산선고 전에 면책을 먼저 허가하는 제도
- **요건**:
  - 재산이 거의 없어 파산절차 진행 실익이 없는 경우
  - 면책불허가 사유가 없는 경우
- **장점**: 파산관재인 선임 불필요 → 비용 절감, 절차 신속

#### 3.4 신속면책 제도
- **대상**: 취약계층 (기초생활수급자, 차상위계층, 한부모가족)
- **협력**: 서울금융복지상담센터와 연계
- **특징**: 신속한 면책 허가

### 4. 상속재산파산 제도

#### 4.1 상속재산파산의 의의
- **목적**: 채무초과 상태의 피상속인 사망 시 상속재산을 청산
- **장점**:
  - 한정승인보다 간편 (법원이 파산관재인 선임 → 전문가 처리)
  - 상속인의 부담 경감

#### 4.2 파산 요건
- **파산원인**: 상속재산이 상속채무를 변제하기에 부족 (채무초과)
- **신청권자**: 상속인, 상속채권자, 유증을 받은 자, 상속재산관리인

#### 4.3 상속재산파산 신청서 (상속인이 신청)
```
상속재산파산 신청서

피상속인
성명: 고(故) ○○○
주민등록번호: 000000-0000000 (사망)
최종 주소: ○○시 ○○구 ○○로 123
사망일: 2024. 5. 15.

신청인 (상속인)
성명: △△△
주민등록번호: 000000-0000000
주소: ○○시 □□구 □□로 456
피상속인과의 관계: 장남

신청 취지

피상속인 고(故) ○○○의 상속재산에 대하여 파산을 선고한다.

라는 결정을 구합니다.

파산 원인

1. 피상속인의 상속재산
   - 부동산: 없음
   - 동산: 가재도구 일체 (시가 약 3,000,000원)
   - 예금: ○○은행 보통예금 (잔액 2,000,000원)
   합계: 약 5,000,000원

2. 상속채무
   - ○○은행 대출금: 200,000,000원
   - △△카드 카드대금: 30,000,000원
   - □□캐피탈 할부금: 20,000,000원
   합계: 250,000,000원

3. 채무초과 상태
   상속재산 5,000,000원으로는 상속채무 250,000,000원을 변제할 수 없으므로
   상속재산파산을 신청합니다.
```

#### 4.4 상속재산파산의 효과
- **상속인의 지위**: 한정승인 간주 (상속재산 범위 내에서만 책임)
- **파산관재인 선임**: 법원이 선임한 파산관재인이 상속재산 관리·청산
- **상속인의 고유재산 보호**: 상속인의 고유재산은 파산재단에 포함되지 않음

### 5. 상속의 승인과 포기

#### 5.1 상속포기
- **의의**: 상속인이 피상속인의 권리·의무를 일체 승계하지 않는 의사표시
- **신고기간**: 상속 개시를 안 날로부터 **3개월 이내** (가정법원 신고)
- **효과**: 처음부터 상속인이 아니었던 것으로 간주 (소급효)

```
상속포기 신고서

성명: △△△
주민등록번호: 000000-0000000
주소: ○○시 □□구 □□로 456

피상속인
성명: 고(故) ○○○
주민등록번호: 000000-0000000 (사망)
사망일: 2024. 5. 15.
신고인과의 관계: 부

신고 취지

신고인은 피상속인 고(故) ○○○의 상속을 포기합니다.

라는 심판을 구합니다.

신고 이유

상속을 포기합니다.

첨부서류

2. 신고인의 가족관계증명서
3. 피상속인의 채무 관련 서류
```

#### 5.2 한정승인
- **의의**: 상속받은 재산의 한도 내에서만 피상속인의 채무를 변제
- **신고기간**: 상속 개시를 안 날로부터 **3개월 이내** (가정법원 신고)
- **효과**: 상속재산을 초과하는 채무는 책임지지 않음

```

```

#### 5.3 특별한정승인
- **요건**:
  - 상속인이 상속재산의 전부 또는 일부를 은닉·처분하는 등 법정단순승인 사유에 해당하는 행위를 한 경우
  - **단, 고의가 아닌 경우** (중대한 과실로 인한 경우 포함)
- **절차**: 가정법원에 특별한정승인 신청
- **효과**: 한정승인의 효력 인정

### 6. 개인회생 제도

#### 6.1 개인회생의 의의
- **목적**: 경제적 어려움으로 파탄에 직면한 개인이 법원의 허가를 받은 변제계획에 따라 채무를 변제하여 경제적 회생 도모
- **파산과의 차이**: 파산은 청산, 회생은 변제계획에 따라 일부 변제 후 나머지 면책

#### 6.2 개인회생 신청 요건
- **채무액**: 무담보 채무 5억원 이하, 담보부 채무 10억원 이하
- **계속적 수입**: 급여, 사업소득 등 정기적 수입이 있어야 함
- **변제 능력**: 변제계획에 따라 최소한의 변제를 할 수 있어야 함

#### 6.3 개인회생 절차
```
개인회생 신청
   ↓
회생절차 개시 결정
   ↓
채권자 이의기간 (채권 확정)
   ↓
변제계획안 제출 및 인가
   ↓
변제 시작 (3년~5년)
   ↓
변제계획 완수
   ↓
면책 결정
```

#### 6.4 변제계획안
- **변제기간**: 3년 (최대 5년 연장 가능)
- **변제율**:
  - **청산가치 보장 원칙**: 파산 시 배당액보다 많이 변제해야 함
  - **가용소득**: 월 소득 - 생계비 = 변제 가능 금액
  - 변제액 = max(청산가치, 가용소득 × 36개월)

### 7. 면책되지 않는 채권 (면책 예외)

- **조세채권**: 국세, 지방세 등
- **벌금, 과료, 형사소송비용**
- **고의·중과실로 가한 불법행위**: 교통사고 음주운전, 폭행 등
- **양육비, 부양료**: 자녀 양육비, 부모 부양료 등
- **근로자 임금**: 채무자의 근로자에 대한 임금, 퇴직금 등
- **학자금대출**: 2023년부터 한국장학재단 학자금대출 원리금 청구권 면책 제외
