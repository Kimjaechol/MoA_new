# 가족관계등록 문서작성 스킬

## 개요

가족관계등록은 개인별 가족관계등록부에 출생, 사망, 혼인, 이혼, 입양 등 신분 변동사항을 기록·관리하는 제도입니다. 2008년 1월 1일 호주제도가 폐지되고 "가족관계의 등록 등에 관한 법률"이 시행되면서, 호주 중심의 가(家) 단위 호적부에서 개인별 가족관계등록부로 전환되었습니다.

이 스킬은 38,331줄의 가족관계등록 실무 데이터베이스를 기반으로 모든 종류의 가족관계등록 신고서를 자동으로 작성합니다.

## 가족관계등록 체계

### 1. 가족관계등록제도의 특징

#### 1.1 개인별 등록부 작성
- **호적제도와의 차이**: 호주 중심 가(家) 단위 → 개인별 등록부
- **등록기준지**: 본인이 자유롭게 선택·변경 가능
- **전산화**: 전산정보처리조직으로 기록·관리
- **국가사무화**: 대법원이 관장

#### 1.2 다양한 증명서 발급
- **가족관계증명서**: 본인과 부모, 배우자, 자녀 관계
- **기본증명서**: 본인의 출생, 사망, 개명 등 기본 사항
- **혼인관계증명서**: 혼인, 이혼 사항
- **입양관계증명서**: 입양, 파양 사항
- **친양자입양관계증명서**: 친양자 입양 사항 (비공개)

### 2. 출생 신고

#### 2.1 신고의무자 및 기간
- **신고의무자**: 부 또는 모, 동거친족, 분만에 관여한 의사·조산사
- **신고기간**: 출생일로부터 1개월 이내
- **지연신고**: 1개월 경과 시 지연사유서 첨부

#### 2.2 신고서 기재사항
```
출생신고서

[신고인 정보]
- 성명, 생년월일, 주소, 본인과의 관계

[출생자 정보]
- 성명, 성별, 출생일시, 출생장소
- 부모 성명 및 등록기준지

[첨부서류]
- 출생증명서 (의사 또는 조산사 작성)
- 신분증 사본
```

#### 2.3 특수한 경우
- **혼인 외 출생**: 모의 성과 본을 따름 (원칙)
- **부모 모두 외국인**: 한국에서 출생 시 한국 국적 취득 불가 (예외: 부모 불명 등)
- **복수출생**: 쌍둥이의 경우 출생 순서 기재

### 3. 사망 신고

#### 3.1 신고의무자 및 기간
- **신고의무자**: 동거친족, 동거자, 사망장소 관리인
- **신고기간**: 사망일로부터 1개월 이내
- **지연신고**: 1개월 경과 시 지연사유서 첨부

#### 3.2 신고서 기재사항
```
사망신고서

[신고인 정보]
- 성명, 주소, 본인과의 관계

[사망자 정보]
- 성명, 주민등록번호
- 사망일시, 사망장소
- 사망원인 (질병사, 사고사, 자살 등)

[첨부서류]
- 사망진단서 또는 사체검안서
- 신분증 사본
```

### 4. 혼인 신고

#### 4.1 신고 요건
- **당사자 합의**: 남녀 간 혼인 의사의 합치
- **법정 혼인연령**: 만 18세 이상
- **혼인 장애사유 없을 것**: 근친혼, 중혼, 직계혈족·인척 간 혼인 금지

#### 4.2 신고서 기재사항
```
혼인신고서

[신고인 정보]
- 부부 쌍방의 성명, 주민등록번호, 주소, 등록기준지

[혼인 정보]
- 혼인성립일자 (신고일 또는 합의일)
- 부부의 등록기준지 결정

[증인]
- 성년자 2인의 증인 서명 또는 기명날인

[첨부서류]
- 가족관계증명서 (쌍방)
- 혼인관계증명서 (쌍방)
- 신분증 사본
```

#### 4.3 국제결혼
- **외국인과의 혼인**: 외국인의 혼인요건구비증명서 제출
- **재외국민**: 재외공관에 혼인신고 가능

### 5. 이혼 신고

#### 5.1 협의이혼
- **가정법원 확인**: 이혼의사 확인 및 양육·재산분할 협의
- **신고기간**: 확인서 받은 날로부터 3개월 이내

```
협의이혼신고서

[신고인 정보]
- 부부 쌍방의 성명, 주민등록번호, 주소

[이혼 정보]
- 이혼성립일자 (신고일)
- 이혼의사 확인서 교부일자

[미성년 자녀]
- 친권자 및 양육자 지정
- 양육비 부담 협의

[첨부서류]
- 이혼의사 확인서 (가정법원)
- 가족관계증명서
- 혼인관계증명서
```

#### 5.2 재판상 이혼
- **판결 확정**: 이혼 판결 확정일로부터 1개월 이내 신고
- **신고의무자**: 소제기자 (원고)

### 6. 입양 신고

#### 6.1 일반 입양
- **입양 요건**:
  - 양친이 성년일 것
  - 양친이 양자보다 연장자일 것 (미성년자 입양의 경우)
  - 배우자 있는 자의 입양은 배우자 동의 필요

```
입양신고서

[양친 정보]
- 성명, 주민등록번호, 주소, 등록기준지

[양자 정보]
- 성명, 주민등록번호 (또는 생년월일)
- 입양 전 성과 본

[입양 정보]
- 입양성립일자
- 친생부모 동의 여부 (미성년자의 경우)

[첨부서류]
- 양자의 가족관계증명서
- 친생부모 동의서 (미성년자의 경우)
- 가정법원 허가서 (미성년자 입양의 경우)
```

#### 6.2 파양 신고
- **협의파양**: 양친·양자 합의 (양자가 미성년자면 가정법원 허가 필요)
- **재판상 파양**: 파양 판결 확정일로부터 1개월 이내 신고

### 7. 친양자 입양 신고

#### 7.1 친양자 제도
- **의의**: 입양을 통해 친생자와 동일한 법적 지위 부여
- **효과**: 친생부모와의 친족관계 단절, 양친의 혼인 중 출생자로 기록

#### 7.2 요건
- **양친**: 3년 이상 혼인 중인 부부가 공동으로 입양
- **양자**: 미성년자
- **가정법원 허가**: 반드시 가정법원의 친양자 입양 허가 필요

```
친양자 입양신고서

[양친 정보]
- 부부 쌍방의 성명, 주민등록번호, 주소

[양자 정보]
- 성명, 생년월일
- 친생부모 정보 (비공개 처리)

[입양 정보]
- 가정법원 심판 확정일자
- 사건번호

[첨부서류]
- 친양자 입양 심판서 정본
- 확정증명서
```

#### 7.3 친양자입양관계증명서
- **비공개**: 본인, 양친, 법원, 검찰 등만 발급 가능
- **친생부모 정보**: 친양자입양관계증명서에만 기재

### 8. 인지 신고

#### 8.1 임의인지
- **신고의무자**: 부 또는 모 (혼인 외 출생자에 대한 인지)
- **동의**: 자가 성년자인 경우 그의 동의 필요

```
인지신고서

[인지자 정보]
- 성명, 주민등록번호, 주소, 등록기준지

[피인지자 정보]
- 성명, 생년월일, 주소
- 모의 성명 (부가 인지하는 경우)

[인지 정보]
- 인지성립일자 (신고일)
- 자의 동의 여부 (성년자의 경우)

[첨부서류]
- 피인지자의 가족관계증명서
- 자의 동의서 (성년자의 경우)
- 모의 동의서 (부가 태아를 인지하는 경우)
```

#### 8.2 강제인지
- **인지청구의 소**: 자가 부 또는 모를 상대로 인지 청구
- **판결 확정**: 확정일로부터 1개월 이내 신고

### 9. 친권 관련 신고

#### 9.1 친권자 지정 신고
- **가정법원 심판**: 친권자 지정 심판 확정
- **신고서 제출**: 확정일로부터 1개월 이내

```
친권자 지정 신고서

[미성년자 정보]
- 성명, 생년월일

[친권자 정보]
- 친권자로 지정된 부 또는 모

[심판 정보]
- 사건번호, 확정일자

[첨부서류]
- 친권자 지정 심판서 정본
- 확정증명서
```

#### 9.2 친권상실 선고 신고
- **가정법원 심판**: 친권상실 선고
- **효과**: 친권 행사 불가

### 10. 후견 관련 신고

#### 10.1 미성년후견인 선임 신고
- **요건**: 친권자 없거나 친권자가 친권 행사 불가능
- **선임**: 가정법원 심판

```
미성년후견인 선임 신고서

[미성년자 정보]
- 성명, 생년월일, 주소

[후견인 정보]
- 성명, 주민등록번호, 주소
- 미성년자와의 관계

[심판 정보]
- 사건번호, 확정일자

[첨부서류]
- 후견인 선임 심판서 정본
- 확정증명서
```

#### 10.2 성년후견 관련 신고
- **성년후견 개시**: 질병, 장애 등으로 사무 처리 능력 부족
- **한정후견**: 일부 사무 처리 능력 부족
- **특정후견**: 특정 사무에 한정

### 11. 실종선고 신고

#### 11.1 실종선고
- **요건**: 부재자의 생사 불명이 일정 기간 계속
  - 보통실종: 5년
  - 특별실종: 1년 (전지, 침몰, 추락 등)

```
실종선고 신고서

[실종자 정보]
- 성명, 주민등록번호
- 최종 주소

[실종 정보]
- 실종 시작일
- 실종 사유 (보통실종 또는 특별실종)

[심판 정보]
- 가정법원 실종선고 심판
- 사건번호, 확정일자
- 사망간주일

[첨부서류]
- 실종선고 심판서 정본
- 확정증명서
```

#### 11.2 실종선고 취소
- **요건**: 실종자 생존 또는 다른 시기 사망 증명
- **효과**: 실종선고 시부터 발생한 법률효과 소급 소멸

### 12. 개명 신고

#### 12.1 개명 허가
- **신청 요건**:
  - 명의 문자가 괴이하거나 난해한 경우
  - 동명이인으로 인한 불편
  - 법률상·사회생활상 현저한 불편
  - 개명할 만한 합당한 사유

```
개명허가 신청서

[신청인 정보]
- 성명, 주민등록번호, 주소, 등록기준지
- 직업, 연락처

[개명 사유]
- 구체적 사유 기재
  예) "본인의 이름이 동명이인이 많아 사회생활에 현저한 불편을 겪고 있습니다."
  예) "이름의 한자가 일반인이 읽기 어려워 생활에 지장이 있습니다."

[변경하고자 하는 이름]
- 개명 전 이름: ○○○
- 개명 후 이름: △△△
- 한자 및 뜻 설명

[소명자료]
- 동명이인 검색 결과 (행정안전부)
- 불편 사례 소명서
- 가족관계증명서, 주민등록등본

[첨부서류]
- 가족관계증명서
- 기본증명서
- 주민등록등본
```

#### 12.2 개명신고
- **가정법원 허가**: 개명허가 심판 확정
- **신고기간**: 확정일로부터 1개월 이내

```
개명신고서

[신고인 정보]
- 성명 (개명 전), 주민등록번호, 주소

[개명 정보]
- 개명 전 이름: ○○○
- 개명 후 이름: △△△

[심판 정보]
- 사건번호, 확정일자

[첨부서류]
- 개명허가 심판서 정본
- 확정증명서
```

### 13. 자녀의 성(姓)과 본(本) 변경 신고

#### 13.1 변경 요건
- **부모 이혼 또는 재혼**: 자녀 복리를 위해 필요한 경우
- **가정법원 허가**: 반드시 가정법원 허가 필요
- **자의 동의**: 자가 13세 이상인 경우 그의 동의 필요

```
성본변경 허가 신청서

[신청인 정보]
- 친권자 또는 후견인 성명, 주소

[자녀 정보]
- 성명, 생년월일
- 현재 성과 본: ○○ ◇◇
- 변경하려는 성과 본: △△ □□

[변경 사유]
- 구체적 사유 기재
  예) "모가 친권자로서 자녀를 양육하고 있으며, 자녀가 모의 성과 본으로 변경하기를 원하고 있습니다. 자녀의 정체성 형성과 학교생활에 도움이 됩니다."

[자의 동의]
- 자가 13세 이상인 경우 동의서 첨부

[첨부서류]
- 가족관계증명서
- 혼인관계증명서
- 자의 동의서 (13세 이상)
```

#### 13.2 성본변경 신고

### 15. 가족관계등록부 정정

#### 15.1 정정 사유
- **기재 착오 또는 누락**: 신고서 기재 오류, 전산 입력 오류
- **기재 누락**: 신고는 되었으나 등록부에 누락

#### 15.2 정정 절차

**① 간이정정 (시·읍·면장 직권)**
- **대상**: 명백한 오기, 탈자 등
- **절차**: 신청서 제출 → 시·읍·면장 직권 정정
- **예**: 한자 오기, 주소 오기

**② 가정법원 허가 정정**
- **대상**: 중요 사항의 정정
- **절차**: 가정법원에 정정허가 신청 → 심판 확정 → 신고

```
가족관계등록부 정정허가 신청서

[신청인 정보]
- 성명, 주민등록번호, 주소, 등록기준지

[정정 대상]
- 정정 대상 등록부: 가족관계증명서 / 기본증명서 / 혼인관계증명서 등
- 정정 대상 인물: 본인 / 부 / 모 / 배우자 / 자녀

[정정 내용]
- 정정 전: ○○○
- 정정 후: △△△

[정정 사유]
- 착오 발생 경위 및 정정 필요성

[소명자료]
- 정정 전후 내용을 입증할 서류
  예) 출생증명서, 혼인신고서, 외국 공문서 등

[첨부서류]
- 가족관계증명서
- 기본증명서
- 소명자료 일체
```
