# 보전처분문서작성 스킬

## 개요
가압류 및 가처분 등 보전처분 신청 문서 작성 스킬입니다.

## 예정된 스킬

### 가압류
- 부동산가압류신청서
- 채권가압류신청서
- 동산가압류신청서
- 자동차가압류신청서
- 선박/항공기가압류신청서

### 가처분
- 부동산처분금지가처분신청서
- 부동산점유이전금지가처분신청서
- 채권가처분신청서
- 영업양도금지가처분신청서
- 이사회소집금지가처분신청서
- 주주총회결의효력정지가처분신청서

### 임시의 지위를 정하는 가처분
- 임금가처분신청서
- 지위보전가처분신청서 (해고 무효)
- 영업계속가처분신청서

### 관련 절차
- 담보제공서
- 담보취소신청서
- 보전이의신청서
- 보전취소신청서
- 보전항고장
