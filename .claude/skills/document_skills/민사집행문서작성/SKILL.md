# 민사집행(강제집행) 문서 작성 스킬

## 개요

민사집행은 확정판결이나 공정증서 등의 집행권원을 바탕으로 채무자의 재산에 대하여 강제적으로 채권을 만족시키는 법원의 절차입니다. 본 스킬은 55,348줄의 민사실무대전 민사집행절차 편 데이터베이스를 기반으로 모든 종류의 민사집행 신청서를 자동으로 작성합니다.

**핵심 기능:**
- **금전채권 강제집행**: 부동산, 선박, 자동차, 중기, 동산, 채권 압류
- **경매 신청**: 강제경매, 임의경매(담보권 실행)
- **배당**: 배당요구, 배당기일, 배당이의
- **인도집행**: 부동산·동산 인도, 명도집행, 철거집행
- **대체집행**: 대체적작위, 부작위의무 위반
- **간접강제**: 부대체적작위, 의사표시 집행
- **구제절차**: 집행이의, 청구이의의 소, 제3자이의의 소, 집행정지

## 문서 용도

민사집행은 다음 상황에서 필수적입니다:

1. **판결 집행**: 승소 판결을 받았으나 채무자가 임의로 이행하지 않는 경우
2. **공정증서 집행**: 강제집행인낙조항이 있는 공정증서로 집행
3. **담보권 실행**: 근저당권자가 부동산 경매를 신청하는 경우
4. **채권 추심**: 채무자의 급여, 예금, 매출채권을 압류하여 추심
5. **부동산 명도**: 건물 점유자를 강제로 퇴거시키는 경우

**활용 사례**: 대여금 회수, 매매대금 회수, 손해배상 집행, 부동산 경매, 임차보증금 회수
