# 행정소송문서작성 스킬

## 개요
행정처분 및 행정작용에 대한 소송 문서 작성 스킬입니다.

## 예정된 스킬

### 항고소송
- 취소소송 소장
- 무효등확인소송 소장
- 부작위위법확인소송 소장

### 당사자소송
- 행정상 당사자소송 소장
- 행정상 부당이득반환청구 소장

### 객관적 소송
- 민중소송 소장
- 기관소송 소장

### 행정심판
- 행정심판청구서 (취소심판)
- 행정심판청구서 (의무이행심판)
- 행정심판청구서 (무효등확인심판)

### 관련 문서
- 집행정지신청서
- 가처분신청서 (행정소송)
- 정보공개청구서
- 이의신청서
