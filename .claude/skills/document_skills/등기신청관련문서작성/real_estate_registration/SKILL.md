# 부동산등기 신청서 작성 스킬

## 개요

부동산등기는 토지와 건물에 대한 권리관계를 공시하는 가장 중요한 법적 절차입니다. 본 스킬은 214,862줄의 부동산등기실무대전 데이터베이스를 기반으로 모든 종류의 부동산등기 신청서를 자동으로 작성합니다.

**핵심 기능:**
- **소유권등기**: 소유권보존, 소유권이전 (매매, 증여, 상속, 교환)
- **담보권등기**: 저당권설정, 근저당권설정, 전세권설정, 질권설정
- **용익권등기**: 지상권설정, 지역권설정, 임차권설정
- **가등기**: 소유권이전청구권, 담보가등기
- **등기말소**: 소유권말소, 저당권말소, 가등기말소
- **변경등기**: 소유자표시변경, 채권최고액변경
- **경정등기**: 등기사항 오류 정정
- **전자신청**: 인터넷등기소 전자신청 자동 생성

## 문서 용도

부동산등기 신청서는 다음 상황에서 필수적입니다:

1. **부동산 거래**: 매매, 증여, 교환으로 소유권이 이전되는 경우
2. **담보 설정**: 대출을 위해 저당권, 근저당권을 설정하는 경우
3. **상속**: 피상속인의 부동산을 상속받은 경우
4. **권리 확보**: 가등기로 장래 권리를 보전하는 경우
5. **등기 정정**: 잘못된 등기 내용을 바로잡는 경우

**활용 사례**: 부동산 매매, 상속, 증여, 대출, 전세, 임대차, 권리분석
