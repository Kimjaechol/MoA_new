# 소유권보존등기 신청서 작성 스킬

## 개요

소유권보존등기란 미등기 부동산에 대하여 최초로 소유권을 등기부에 기재하는 등기입니다. 부동산등기법 제65조에서 소유권보존등기를 신청할 수 있는 자를 규정하고 있으며, 본 스킬은 부동산등기실무대전을 기반으로 토지 및 건물의 소유권보존등기 신청서를 정확하게 작성합니다.

**핵심 특징:**
- 단독신청: 등기권리자만의 신청으로 등기 가능 (공동신청 불필요)
- 등기원인 불기재: 보존등기는 등기원인과 그 연월일을 기재하지 않음
- 신청근거규정 기재: 부동산등기법 제65조 각 호 중 해당 규정 기재 필수

## 소유권보존등기를 신청할 수 있는 자 (부동산등기법 제65조)

### 제1호: 대장에 최초 소유자로 등록된 자 또는 그 상속인

```python
신청자격_제1호 = {
    "토지": {
        "신청자": "토지대장 또는 임야대장에 최초의 소유자로 등록되어 있는 자",
        "상속인": "대장상 최초 소유자의 상속인 (포괄승계인 포함)",
        "법인합병": "대장상 소유자인 법인이 합병으로 소멸한 경우 존속법인",
        "법인분할": "분할 전 법인이 소멸한 경우 분할된 법인"
    },
    "건물": {
        "신청자": "건축물대장에 최초의 소유자로 등록되어 있는 자",
        "상속인": "대장상 최초 소유자의 상속인",
        "구분건물": "집합건축물대장에 최초의 소유자로 등록되어 있는 자"
    }
}
```

**등기선례:**
- 대장상 소유자의 성명, 주소 등의 일부 누락 또는 착오가 있어 대장상 소유자표시를 정정 등록한 경우도 포함 (등기예규 제1483호)
- 미등기 토지의 양수인은 대장상 소유권이전등록이 되어 있더라도 직접 자기명의로 소유권보존등기 불가 → 최초 소유자 명의로 보존등기 후 순차로 이전등기 필요

### 제2호: 확정판결에 의하여 자기의 소유권을 증명하는 자

```python
신청자격_제2호 = {
    "적용대상": "소유권확인판결을 받은 자",
    "판결유형": [
        "원고 소유임을 확인하는 판결",
        "피고 명의 소유권보존등기의 말소를 명하는 판결"
    ],
    "첨부서류": [
        "확정판결정본",
        "확정증명서",
        "송달증명서 (패소자에게 송달된 경우)"
    ],
    "특이사항": "대장상 소유자와 무관하게 판결로 소유권 증명 가능"
}
```

### 제3호: 수용으로 인하여 소유권을 취득한 자

```python
신청자격_제3호 = {
    "적용대상": "공익사업을 위한 토지 등의 취득 및 보상에 관한 법률에 따른 수용",
    "첨부서류": [
        "재결서 정본",
        "토지(건축물)대장등본",
        "보상금 지급 또는 공탁 증명서"
    ]
}
```

### 제4호: 특별자치도지사, 시장, 군수 또는 구청장의 확인서

```python

    "적용대상": "무허가건물 등 대장에 등록되지 않은 건물",
    "첨부서류": [
        "시장·군수·구청장의 확인서",

    ],
    "특이사항": "건축물대장 등록 없이 보존등기 가능"
}
```

```
  주    소   서울특별시 서초구 서초로120길 10
```

**첨부서류:**
```python
건물_보존등기_첨부서류 = [
```
