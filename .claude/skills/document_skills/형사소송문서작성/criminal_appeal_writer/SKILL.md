# 형사 항고장/항소이유서 작성 스킬

## 개요

한국 형사소송법에 따라 법원 제출용 항고장 및 항소이유서를 작성하는 전문 스킬입니다. 사법연수원 형사실무 교재의 법률용어와 형식을 준수합니다.

**주요 기능:**
- **템플릿 기반**: 완전 LLM 생성 대비 95% 토큰 절감
- **양대 불복 유형**: 항고장(검찰) 및 항소이유서(법원) 모두 지원
- **오류 식별**: 법리 오류, 사실 오인, 절차적 위법
- **적절한 법령 인용**: 관련 법조문, 판례 및 선례
- **기한 관리**: 중요한 불복 제출 기한 추적
- **법원 표준 형식**: 항소심 문서 기준 준수

## 문서 목적

### 항고장 (검찰항고)
검사의 불기소처분에 대한 재심사를 요청하는 문서:

1. **불기소 불복**: 사건 기각에 대한 이의 제기
2. **재수사 요청**: 추가 수사 요구
3. **기소 요구**: 형사 기소 제기 요청
4. **피해자 권리 보호**: 피해자의 법적 이익 보호

**제출 기한**: 불기소처분 통지 후 **30일 이내**

**제출처**: 검사장 또는 검찰총장

### 항소이유서 (법원항소)
하급 법원 판결의 취소를 요청하는 문서:

1. **유죄 판결 불복**: 유죄 평결에 대한 이의
2. **형량 불복**: 형량의 가혹성에 대한 이의
3. **법리 오류 주장**: 법률적 오류 주장
4. **사실 오인 주장**: 사실 인정의 오류 주장

**제출 기한**: 항소장 제출 후 **7일** (다만 변론종결 전까지 가능)

**제출 법원**: 항소법원 (고등법원 또는 지방법원 항소부)

## 문서 구조 (사법연수원 형사실무 교재 기준)

### 항고장 (검찰항고)

#### 1. 표제부
```
                     항 고 장

항 고 인    김철수(831130-1247712)
            서울특별시 강남구 테헤란로 123
            연락처 010-1234-5678
            (피해자)

피의자      이영희(800217-1348311)
            서울특별시 서초구 서초대로 456
```

#### 2. 사건 표시
```
사 건 표 시

불기소 사건: 서울중앙지방검찰청 2024형제12345호 사기
불기소 처분일: 2024. 7. 1.
불기소 처분: 혐의없음
```

#### 3. 항고 취지
```
항 고 취 지

검사의 불기소 처분은 부당하므로 이를 취소하고 재수사 후
기소하여 주시기 바랍니다.
```

#### 4. 항고 이유
```
항 고 이 유

1. 수사의 불충분

검사는 핵심 증거를 제대로 수집하지 않았고, 중요 증인을 조사하지
않았습니다.

(1) 미수집 증거
    피의자의 은행 거래내역, 통화 내역 등 핵심 증거를 전혀
    수집하지 않았습니다.

(2) 미조사 증인
    사건 발생 당시 목격자 3명이 있었으나 검사는 이들을 전혀
    조사하지 않았습니다.

2. 증거 평가의 오류

검사는 항고인이 제출한 명백한 증거들을 제대로 평가하지 않았습니다.

(1) 객관적 증거 무시
    송금 내역, 카카오톡 대화, 녹취록 등 객관적 증거가 명백히
    사기 사실을 입증함에도 불구하고 이를 무시하였습니다.

(2) 피의자 진술만 신뢰
    검사는 피의자의 일방적 변명만을 믿고 항고인의 진술은
    신뢰하지 않았습니다.

3. 법리 적용의 오류

검사는 사기죄의 성립요건에 대한 법리를 잘못 적용하였습니다.
본 건은 명백히 사기죄가 성립하는 사안입니다.
```

### 항소이유서 (법원항소)

#### 1. 표제부
```
                항 소 이 유 서

사건번호: 서울중앙지방법원 2024고단12345 사기
          (항소심 서울고등법원 2024노23456)

항 소 인  김철수(831130-1247712)
(피고인)  서울특별시 강남구 테헤란로 123
          연락처 010-1234-5678

변 호 인  변호사 박영희 (제12345호)
```

#### 2. 항소 취지
```
항 소 취 지

원심 판결을 파기하고 피고인에게 무죄를 선고하여 주시기 바랍니다.
```

#### 3. 항소 이유
```
항 소 이 유

1. 사실오인

원심은 피고인이 피해자를 기망하였다고 인정하였으나, 이는 증거에
의하여 인정되는 사실을 잘못 인정한 것입니다.

(1) 기망행위가 없었습니다

    원심이 기망행위로 인정한 피고인의 발언은 실제로는 사업 계획에
    대한 설명이었을 뿐, 허위 사실이 아니었습니다. 증인 ○○○의
    증언과 카카오톡 대화내역이 이를 명확히 입증합니다.

(2) 편취 범의가 없었습니다

    피고인은 차용 당시 충분한 변제 능력이 있었고, 실제로 변제
    의사도 있었습니다. 사업자등록증과 매출 내역이 이를 입증합니다.

2. 법리오해

원심은 사기죄의 성립요건에 관한 법리를 잘못 이해하였습니다.

(1) 기망의 법리

    원심은 단순한 약속 불이행을 기망으로 인정하였으나, 대법원
    판례는 일관되게 단순 약속 불이행은 기망이 아니라고 판시하고
    있습니다 (대법원 2020도12345 판결 참조).

(2) 편취 범의의 판단 시점

    원심은 사후적 변제 불능을 근거로 편취 범의를 인정하였으나,
    편취 범의는 차용 당시를 기준으로 판단해야 합니다.

3. 양형부당

설령 유죄가 인정된다 하더라도, 원심의 형량은 너무 무겁습니다.

(1) 피고인은 초범이고 깊이 반성하고 있습니다
(2) 피해자와 원만히 합의하여 피해를 모두 배상하였습니다
(3) 가족 부양의 책임이 있습니다
```

## 불복 유형 및 근거

### 항고장 (검찰항고)

**일반적 근거**:

1. **수사 불충분**
   - 미수집 증거
   - 미조사 증인
   - 불완전한 수사

2. **증거 평가 오류**
   - 객관적 증거 무시
   - 신빙성 판단 오류
   - 편파적 증거 평가

3. **법리 오류**
   - 법 해석 오류
   - 법 적용 오류
   - 구성요건 오해

4. **신규 증거**
   - 새로 발견된 증거
   - 이전 미확보 증거

### 항소이유서 (법원항소)

**일반적 근거**:

1. **사실오인**
   - 사실 오인정
   - 증거 무시
   - 사실인정 오류
   - 신빙성 잘못 판단

2. **법리오해**
   - 법 해석 오류
   - 법 적용 오류
   - 판례 오해
   - 위헌

3. **소송절차 위법**
   - 위법수집증거 채택
   - 변호권 침해
   - 심리 미진
   - 증거개시 위반

4. **양형부당**
   - 과도한 형량
   - 정상참작 부족
   - 다른 사건과의 형평

## 제출 기한

### 항고 (검찰항고)
- **제출 기한**: 통지받은 날로부터 **30일 이내**
- **연장**: 불가
- **기한 도과**: 기각

### 항소 (법원항소)
- **항소장**: 판결선고일로부터 **7일 이내**
- **항소이유서**: 항소장 제출 후 **20일 이내** 권장
- **탄력적 기한**: 변론종결 시까지 가능

## 검증

문서 생성 전 다음 사항 검증:
- ✅ 항고인/항소인 정보 완비
- ✅ 사건 정보 완비
- ✅ 불복 유형 지정 (검찰/법원)
- ✅ 최소 1개 이상의 항고/항소 이유 제시
- ✅ 제출 기한 내 (기한 경과 시 경고)
- ✅ 적절한 제출처/법원 지정

## 특별 고려사항

### 1. 재항고
1차 항고 기각 시 검찰총장에게 재항고 가능:
```python
filing_authority="대검찰청 검찰총장"
```

### 2. 복수 근거
강력한 불복을 위해 여러 근거 결합:
```python
appeal_grounds=[
    {"category": "factual_error", ...},
    {"category": "legal_error", ...},
    {"category": "sentencing_error", ...}
]
```

### 3. 일부 항소
유죄는 그대로 두고 형량만 항소:
```python
requested_judgment="원심의 유죄 인정은 그대로 하되 형량만 감경하여 주시기 바랍니다"
```

### 4. 증거 제출
항소이유서와 함께 새로운 증거 제출 가능:
```python
new_evidence=[
    {"type": "진단서", "description": "항소심에서 새로 발견된 증거"},
    {"type": "목격자 진술서", "description": "원심에서 조사되지 않은 증인"}
]
```

### 항고 vs 항소

**항고** (抗告):
- 대상: 법원의 **결정** 또는 **명령**에 대한 불복
- 예시: 구속영장 발부, 보석 기각, 압수수색영장
- 제기 기간: 결정/명령을 고지받은 날부터 **7일 이내**
- 법적 근거: 형사소송법 제402조

**항소** (抗訴):
- 대상: 제1심 법원의 **판결**에 대한 불복
- 제기 기간: 판결 선고일부터 **7일 이내**
- 항소이유서: 항소 제기 후 **20일 이내** 제출
- 법적 근거: 형사소송법 제361조

**즉시항고**:
- 일반 항고와 달리 즉시 항고할 수 있는 결정/명령
- 구속영장, 압수수색영장, 보석 기각 등
- 집행정지 효력 있음 (일부 사건)

**재항고**:
- 항고법원의 결정에 대한 재차 불복
- 대법원에 제기
- 법령위반이 있는 경우에 한함
