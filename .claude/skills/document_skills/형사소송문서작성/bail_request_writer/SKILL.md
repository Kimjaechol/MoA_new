# 보석허가청구서 작성 스킬

## 개요

한국 형사소송법에 따라 법원 제출용 보석허가청구서를 작성하는 전문 스킬입니다. 사법연수원 형사실무 교재의 법률용어와 형식을 준수합니다.

**주요 기능:**
- **템플릿 기반**: 완전 LLM 생성 대비 95% 토큰 절감
- **양대 보석 유형**: 임의적 보석과 권리보석 모두 지원
- **도주 염려 평가**: 도주 및 증거인멸 위험 종합 평가
- **보석금액 산정**: 적정 보증금 계산 및 근거 제시
- **석방 조건**: 조건부 석방 조건 제안
- **법원 표준 형식**: 형사소송 절차 기준 준수

## 문서 목적

보석허가청구서는 구속된 피고인의 재판 전 석방을 요청하는 공식 문서입니다:

1. **석방 요청**: 재판 전 석방 신청
2. **보석금 제시**: 보증금 납입 제안
3. **도주 방지**: 도주 위험 완화 방안 제시
4. **권리 보전**: 방어권 준비를 위한 석방 필요성
5. **조건 제시**: 석방 조건 제안

**법적 근거**: 형사소송법 제94조~제101조

**제출 법원**: 사건 계속 법원

**성공률**: 범죄 유형 및 정황에 따라 40-60%

## 문서 구조 (사법연수원 형사실무 교재 기준)

### 1. Header (표제부)
```
                보석허가청구서

청 구 인    김철수(831130-1247712)
            서울특별시 강남구 테헤란로 123
            연락처 010-1234-5678
            현재 서울구치소 수용 중
```

### 2. Case Information (사건 표시)
```
사 건 표 시

피고인: 김철수
사건번호: 서울중앙지방법원 2024고단12345 사기
구속일: 2024. 7. 1.
```

### 3. Request Purpose (청구 취지)
```
청 구 취 지

청구인에 대하여 보석금 금 50,000,000원을 납입하는 조건으로
보석을 허가하여 주시기 바랍니다.
```

### 4. Request Reasons (청구 이유)
```
청 구 이 유

1. 보석의 요건 충족

청구인의 경우 형사소송법 제95조가 정한 보석 불허 사유에 해당하지
않으며, 다음과 같은 사정으로 보석이 필요하고 상당합니다.

(1) 도망의 염려가 없습니다
    청구인은 서울시 강남구에 20년간 거주하여 왔으며, 배우자 및
    자녀와 함께 생활하고 있어 도망할 염려가 전혀 없습니다.
    또한 여권을 법원에 제출하였고, 출국금지 조치도 되어 있어
    국외 도피의 가능성도 없습니다.

(2) 증거인멸의 염려가 없습니다
    이미 수사기관이 모든 증거를 확보하였고, 피해자들도 모두
    조사가 완료되어 청구인이 증거를 인멸할 여지가 없습니다.

(3) 재범의 염려가 없습니다
    청구인은 이번이 처음이자 마지막 범죄로, 깊이 반성하고
    있으며 재범의 염려가 전혀 없습니다.

2. 보석의 필요성

(1) 변호권 행사를 위한 필요
    청구인은 혐의를 부인하고 있으며, 효과적인 방어권 행사를
    위해서는 석방 상태에서 변호인과 충분한 상의를 하고 증거를
    수집할 필요가 있습니다.

(2) 가족 부양의 필요
    청구인은 가족의 유일한 생계부양자로서, 계속 구금될 경우
    70세 노모와 어린 자녀 2명의 생계가 막막한 상황입니다.

(3) 건강상의 이유
    청구인은 당뇨병으로 인슐린 투여가 필요하나, 구치소에서는
    적절한 치료를 받기 어려운 실정입니다.

3. 보석 조건 제시

청구인은 다음과 같은 조건을 이행할 것을 서약합니다:

(1) 보석금 금 50,000,000원을 납입하겠습니다
(2) 주거지를 이탈하지 않겠습니다
(3) 소환에 불응하지 않겠습니다
(4) 증거를 인멸하지 않겠습니다
(5) 피해자나 증인과 접촉하지 않겠습니다
```

### 5. Evidence (소명방법)
```
소 명 방 법

1. 갑 제1호증    주민등록등본
2. 갑 제2호증    가족관계증명서
3. 갑 제3호증    재직증명서

```

## 보석의 종류

### 1. 임의적 보석

**형사소송법 제94조**: 법원의 재량에 따른 보석 허가

**허가 요인**:
- 도주 우려 없음
- 증거인멸 우려 없음
- 비폭력 범죄
- 초범
- 건강 문제
- 가족 부양
- 방어권 준비 필요

### 2. 권리보석

**형사소송법 제95조**: 특정 제외 사유가 없으면 법원이 반드시 허가

**허가 요건**:
- 최고형이 징역 3년 이하
- 일정 기간 이상 구금
- 도주/증거인멸 우려 없음

**보석 불허 사유**:
- 상습범
- 도주 우려
- 증거인멸 우려
- 피해자 위해 우려

## 보석금액 산정 기준

### 산정 요소

**범죄의 중대성**:
- 경미한 범죄: 500만원~1,000만원
- 중간 범죄: 1,000만원~3,000만원
- 중대 범죄: 3,000만원~1억원
- 매우 중대: 1억원 이상

**개인적 사정**:
- 경제력
- 도주 위험성
- 전과
- 사회적 유대

**범죄별 일반적 금액**:
```python
bail_amounts = {
    "fraud_under_50m": 10000000,      # 5천만원 미만 사기
    "fraud_50m_to_300m": 30000000,    # 5천만원~3억 사기
    "fraud_over_300m": 50000000,      # 3억원 이상 사기
    "embezzlement": 30000000,         # 횡령
    "assault": 5000000,               # 폭행/상해
    "drug_possession": 20000000,      # 마약 소지
    "drunk_driving": 3000000          # 음주운전
}
```

## 보석 사유

### 1. 도망 염려 없음
- 고정 주소
- 장기 거주
- 가족 관계
- 안정된 직장
- 여권 제출
- 출국금지
- 해외 연고 없음

### 2. 증거인멸 염려 없음
- 증거 확보 완료
- 디지털 증거 보전
- 증인 조사 완료
- 피해자 확보
- 물적 증거 수집

### 3. 가족 부양
- 유일한 생계부양자
- 노부모 부양
- 어린 자녀
- 장애 가족
- 가족 위급상황

### 4. 건강 문제
- 중증 질환
- 만성 질환
- 약물 치료 필요
- 수술 필요
- 교정시설 치료 부족

### 5. 방어권 행사
- 변호인 상담 필요
- 증거 수집 필요
- 전문가 자문 필요
- 복잡한 사건

형사소송법 제94조~제101조에 따라 구속된 피고인이 보증금을 납부하고 석방을 요청하는 제도입니다.

**보석의 종류**:
1. **권리보석** (형사소송법 제95조): 법정형이 사형, 무기 또는 장기 10년을 초과하는 징역이나 금고에 해당하지 않는 사건
2. **임의보석** (형사소송법 제96조): 법원의 재량에 따른 보석
3. **직권보석** (형사소송법 제97조): 법원의 직권 보석

**보석 허가 기준**:
- 도주 염려가 없을 것
- 증거 인멸 염려가 없을 것
- 출석의무 이행 가능성

**보석보증금**: 법원이 정하는 금액 (통상 500만원~5,000만원)
