# 형사소송문서작성 스킬

## 개요
형사소송 절차에서 사용되는 모든 법률 문서 작성 스킬입니다.

## 포함될 스킬

### 수사 단계
- **형사 고소장 (criminal_complaint_writer)** ✅ - 범죄 고소
- **탄원서 (petition_writer)** ✅ - 피해자/참고인 의견서
- **합의서** 🔜 - 피해자-가해자 간 합의

### 구속 관련
- **구속적법여부심사청구서 (arrest_review_request_writer)** ✅ - 구속 적법성 심사
- **보석허가청구서 (bail_request_writer)** ✅ - 보석 허가 신청

### 공판 단계
- **변론요지서 (trial_summary_writer)** ✅ - 변론 요지 정리

### 불복 절차
- **형사 항고장 (criminal_appeal_writer)** ✅ - 결정/명령에 대한 불복
- **항소이유서** 🔜 - 제1심 판결 불복

### 형사사건 분석 스킬 (독립)
- **Crime Requirements Analyzer** ✅ - 범죄구성요건 분석
- **Crime Theory Analyzer** ✅ - 죄수론 분석

## 상태
- 구현 완료: 9개 문서 작성 스킬 + 2개 분석 스킬 ✅
합의서작성 skill, 항고장작성 skill, 재심청구서작성 skill도 추가됨
