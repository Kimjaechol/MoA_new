# 탄원서/진정서 작성 스킬

## 개요

한국 형사소송법에 따라 검찰, 경찰 또는 법원 제출용 탄원서 및 진정서를 작성하는 전문 스킬입니다. 사법연수원 형사실무 교재의 법률용어와 형식을 준수합니다.

**주요 기능:**
- **양대 탄원 유형**: 탄원서(선처요청)와 진정서(신고)
- **템플릿 기반**: 완전 LLM 생성 대비 95% 토큰 절감
- **관계 기술**: 피고인/피해자와의 관계 적절히 기술
- **다양한 탄원 사유**: 개인 사정, 가족 부양, 초범 등
- **전문적 형식**: 탄원 문서 사법 기준 준수
- **자동 서식**: 법원 제출용 DOCX/PDF 출력

## 문서 목적

### 탄원서 (선처 요청)
가족, 친구, 고용주 또는 피해자가 형사 기소 또는 양형에서 관용을 요청하는 문서:

1. **관용 요청**: 자비로운 처분 요청
2. **정상 참작**: 참작할 사정 제시
3. **인품 증명**: 좋은 품성 증언
4. **합의 표명**: 피해자와의 합의 (해당 시)

**제출처**: 검찰청, 법원

### 진정서 (당국 신고)
부정행위를 신고하거나 당국에 조사를 요청하는 문서:

1. **부정 신고**: 불법 행위 또는 비리 신고
2. **조사 요청**: 철저한 조사 요청
3. **시정 요구**: 시정 조치 요구
4. **공익 보호**: 공공의 이익 보호

**제출처**: 경찰서, 검찰청, 정부 기관

## 문서 구조 (사법연수원 형사실무 교재 기준)

### 탄원서 (Petition for Leniency)

#### 1. Header (표제부)
```
                     탄 원 서

탄원인    김철수(831130-1247712)
          서울특별시 강남구 테헤란로 123
          연락처 010-1234-5678
          피고인과의 관계: 직장 동료

피고인    이영희(800217-1348311)
          서울특별시 서초구 서초대로 456
```

#### 2. Petition Purpose (탄원 취지)
```
탄 원 취 지

탄원인은 피고인에 대하여 다음과 같은 사유로 관용 있는 처분을
구하고자 탄원서를 제출합니다.
```

#### 3. Petition Reasons (탄원 사유)
```
탄 원 사 유

1. 피고인은 평소 성실하고 책임감 있는 사람으로, 이번 일은 일시적인
   판단 착오로 인한 것입니다.

2. 피고인은 70세 노모와 어린 자녀 2명을 부양하고 있어, 엄중한 처벌을
   받을 경우 가족이 생계에 어려움을 겪게 됩니다.

3. 피고인은 깊이 반성하고 있으며, 피해자와 원만히 합의하여 피해를
   모두 배상하였습니다.

4. 피고인은 이번이 처음이자 마지막 범죄이며, 다시는 같은 잘못을
   반복하지 않겠다고 다짐하고 있습니다.
```

#### 4. Closing (결어)
```
위와 같은 사정을 참작하시어 피고인에게 관용 있는 처분을 내려주시기를
간곡히 탄원합니다.
```

#### 5. Date and Signature (날짜 및 서명)
```
2024.  7.  15.

                     탄원인  김 철 수  (인)


서울중앙지방검찰청   귀중
```

### 진정서 (Complaint to Authorities)

#### 1. Header (표제부)
```
                     진 정 서

진정인    김철수(831130-1247712)
          서울특별시 강남구 테헤란로 123
          연락처 010-1234-5678

피진정인  이영희(800217-1348311)
          서울특별시 서초구 서초대로 456
          소속: ○○회사 영업팀장
```

#### 2. Complaint Purpose (진정 취지)
```
진 정 취 지

진정인은 피진정인의 부당한 행위에 대하여 철저한 조사와
시정 조치를 요청하고자 진정서를 제출합니다.
```

#### 3. Complaint Details (진정 내용)
```
진 정 내 용

1. 피진정인은 2024. 5. 1.경 직장 내에서 진정인에게 업무와 무관한
   심부름을 강요하며 인격적 모욕을 주었습니다.

2. 피진정인은 진정인이 이를 거부하자 부당한 인사 불이익을 주겠다고
   협박하였습니다.

3. 이러한 행위는 근로기준법 및 직장 내 괴롭힘 금지 규정을 위반하는
   것입니다.

4. 진정인은 정신적 고통을 받았으며, 정상적인 업무 수행이 어려운
   상황입니다.
```

## 탄원서 유형

### 1. 탄원서 (선처 요청)

**일반적 관계**:
- 가족 (Family): 부모, 배우자, 자녀, 형제자매
- 직장 관계 (Work): 동료, 상사, 부하직원
- 사회 관계 (Social): 친구, 이웃, 지인
- 피해자 (Victim): 피해자 본인 또는 가족

**일반적 선처 사유**:
- 초범
- 깊은 반성
- 피해자와 합의
- 가족 부양
- 평소 좋은 인품
- 건강 문제
- 직장 업무
- 고령 또는 연소

### 2. 진정서 (당국 신고)

**일반적 신고 유형**:
- 직장 내 괴롭힘
- 부당 해고
- 소비자 피해
- 공무원 부조리
- 경찰 부당 행위
- 환경 위반
- 건축 위반
