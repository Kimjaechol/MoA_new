# 가사소송 문서 작성 스킬

## 개요

가사소송은 가족관계에 관한 분쟁을 법원에서 해결하는 절차입니다. 본 스킬은 103,983줄의 가사소송서식 데이터베이스를 기반으로 모든 종류의 가사소송 문서를 자동으로 작성합니다.

**핵심 기능:**
- **이혼 소송**: 유책이혼, 재산분할, 위자료, 양육권·친권 지정
- **양육권 분쟁**: 친권행사자 지정, 양육자 변경, 양육비 청구
- **면접교섭권**: 면접교섭 허가, 제한, 변경
- **인지 청구**: 혼인외 출생자의 친자관계 확인
- **친생부인**: 친생자관계 부존재 확인
- **상속**: 한정승인, 재산포기, 유언무효 확인
- **가사보전처분**: 접근금지, 부동산 가압류

## 문서 용도

가사소송은 다음 상황에서 필수적입니다:

1. **이혼**: 배우자의 부정행위, 가혹행위, 유기 등으로 혼인 파탄
2. **양육권**: 자녀 양육권 분쟁, 양육비 미지급
3. **재산분할**: 이혼 시 혼인 중 형성한 재산 분할
4. **인지**: 혼인하지 않은 상태에서 출생한 자녀의 친자관계 확인
5. **상속**: 상속재산이 채무 초과 시 한정승인 또는 포기

**활용 사례**: 이혼, 양육비, 재산분할, 위자료, 면접교섭, 상속포기
