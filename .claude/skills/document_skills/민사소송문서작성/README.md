# 민사소송문서작성 스킬

## 개요
민사소송 절차에서 사용되는 모든 법률 문서 작성 스킬입니다.

## 포함될 스킬 (17개)

### 소송 개시 단계
- **소장 (complaint_writer)** - 민사소송 제기
- **답변서 (answer_writer)** - 피고의 답변
- **반소장 (counterclaim_writer)** - 피고의 반대 청구

### 소송 진행 단계
- **준비서면 (brief_writer)** - 주장 및 증거 제출
- **서증제출서 (evidence_submission_form)** - 서면 증거 제출
- **증거설명서 (evidence_description)** - 증거의 취지 및 관련성 설명
- **증인신문신청서 (witness_examination_form)** - 증인 신문 신청
- **증인신문사항 (direct_examination_matters)** - 주신문 질문 작성 (NEW)
- **반대신문사항 (cross_examination_matters)** - 반대신문 질문 작성 (NEW)
- **감정신청서 (expert_examination)** - 전문가 감정 신청
- **검증신청서 (inspection_request)** - 현장 검증 신청
- **문서제출명령신청서 (document_production_order)** - 상대방 문서 제출 명령
- **증거보전신청서 (evidence_preservation_request)** - 증거 보전 신청

### 소송 절차 관리
- **당사자표시정정신청서 (party_correction)** - 당사자 정보 수정
- **청구변경신청서 (claim_modification)** - 소송 청구 내용 변경
- **일반신청서 (motion_writer)** - 기타 신청

### 불복 절차
- **항소장 (appeal_writer)** - 제1심 판결 불복
- **상고장 (cassation_writer)** - 제2심 판결 불복

## 상태
현재 17개 스킬 모두 구현 완료 ✅

## 최근 추가 (v5.12.0)
- **증인신문사항 (direct_examination_matters)** - 전략적 주신문 질문 작성, 의료소송 특화
- **반대신문사항 (cross_examination_matters)** - 효과적인 반대신문 질문 작성, 증언 탄핵 전략
